# Omniflow Design Style 7: Data Forensics

**Cocok untuk:** Analytics Shopee, spreadsheet analysis, marketplace insights, audit data.

## Creative Thesis

Insight tidak muncul dari dashboard cantik. Ia ditemukan dengan membedah data mentah, menandai pola, dan menghubungkan bukti.

## Signature Device

**The Selection Box:** outline cell aktif berpindah antar-slide, membesar menjadi chart window, customer segment, basket rule, lalu decision frame.

## Palette

- Sheet White `#F8F9F5`
- Grid Gray `#D9DDD6`
- Formula Green `#1F8F5F`
- Ultraviolet `#643DFF`
- Evidence Orange `#FF7A2F`
- Ink `#171A1D`

## Typography

- Display: Space Grotesk
- Brand/body: Outfit
- Formula/data: IBM Plex Mono

## Material Grammar

- spreadsheet cells;
- formula bars;
- row and column coordinates;
- selection handles;
- evidence pins;
- chart fragments;
- analyst annotations.

## Scene Types

1. Raw File Drop
2. Column Normalizer
3. Metric Extraction
4. Time Heatmap
5. Geographic Trace
6. RFM Segmentation
7. Basket Association
8. Analyst Verdict

## Narrative Sequence

`UPLOAD → CLEAN → AGGREGATE → LOCATE → SEGMENT → ASSOCIATE → INTERPRET → DECIDE`

## Avoid

- Dashboard screenshot memenuhi semua slide.
- Neon hacker aesthetic.
- AI sparkle icons.
- Chart tanpa unit dan konteks.
- Angka simulasi tampil sebagai benchmark nyata.

## QA

- [ ] Raw data terlihat minimal sekali.
- [ ] Selection box berubah fungsi minimal tiga kali.
- [ ] Maksimal tiga chart penuh dalam satu carousel.
- [ ] Setiap insight memiliki sumber visual.
- [ ] Tidak menampilkan data seller nyata.
