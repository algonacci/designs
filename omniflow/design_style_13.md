# Omniflow Design Style 13: Archive to Atlas

**Cocok untuk:** DMS, KMS, knowledge management, enterprise search, organizational memory.

## Creative Thesis

File yang rapi belum menjadi pengetahuan. Desain bergerak dari arsip fisik menuju jaringan makna yang dapat ditemukan dan digunakan kembali.

## Signature Device

**The File Tab:** tab folder berpindah antar-slide, berubah menjadi version tab, category, citation, knowledge node, dan semantic connection.

## Palette

- Archive Beige `#D8C9AE`
- Library Green `#294E40`
- Citation Yellow `#F2D34F`
- Burgundy `#7A3145`
- Knowledge Blue `#315CDB`
- Paper `#F6F2E8`

## Typography

- Display: Newsreader
- Brand/body: Outfit
- Index/citation: IBM Plex Mono

## Material Grammar

- folder tabs;
- index cards;
- archive labels;
- version stacks;
- citation markers;
- semantic constellation;
- marginal notes.

## Scene Types

1. Lost File Cabinet
2. Workspace Index
3. Version Stack
4. Permission Drawer
5. Knowledge Extraction
6. Review Desk
7. Semantic Atlas
8. Organizational Memory Map

## Narrative Sequence

`STORE → CLASSIFY → VERSION → CONTROL → EXTRACT → REVIEW → CONNECT → RETRIEVE`

## Avoid

- Folder icon grids.
- Cloud storage illustration.
- AI chat sebagai satu-satunya manfaat KMS.
- Semua node dihubungkan garis acak.
- Citation tanpa sumber visual.

## QA

- [ ] DMS dan KMS memiliki material berbeda tetapi tersambung.
- [ ] File Tab berubah menjadi knowledge node.
- [ ] Version history terlihat sebelum semantic layer.
- [ ] Search menunjukkan makna, bukan hanya keyword.
- [ ] Tidak ada dokumen internal nyata.
