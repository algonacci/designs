# Omniflow Design Style 11: Learning Anatomy

**Cocok untuk:** LMS, adaptive learning, assessment, learner insights.

## Creative Thesis

Belajar adalah sistem berlapis. Jawaban menjadi skor, skor menjadi pola, pola menjadi gap, lalu gap membentuk langkah belajar berikutnya.

## Signature Device

**Insight Scan:** bidang scan bergerak melalui lapisan transparan dan mengungkap response, score, pattern, gap, recommendation, dan learning path.

## Palette

- Vellum Ivory `#F2EEE4`
- Deep Aubergine `#2D173A`
- Insight Lime `#C7F34A`
- Neural Coral `#FF6658`
- Scan Blue `#3568FF`
- Graphite `#25252C`

## Typography

- Display: DM Sans Black dan italic editorial accent
- Brand/body: Outfit
- Assessment data: IBM Plex Mono

## Material Grammar

- vellum sheets;
- registration marks;
- contour maps;
- cross-section diagrams;
- scan beam;
- translucent annotations;
- learning pathway nodes.

## Scene Types

1. Question Void
2. Score Layer
3. Answer Anatomy
4. Personality Contour
5. Learning Tissue
6. Gap Scan
7. Adaptive Route
8. Resolved Learning Map

## Narrative Sequence

`ANSWER → SCORE → PATTERN → GAP → INSIGHT → ADAPT → LEARN → REASSESS`

## Avoid

- Brain icon.
- Robot tutor illustration.
- Semua slide berupa course cards.
- AI glow tanpa informasi.
- Klaim personalisasi tanpa mekanisme.

## QA

- [ ] Insight Scan membuka informasi secara bertahap.
- [ ] Skor bukan focal point akhir.
- [ ] Learning path memperlihatkan perubahan nyata.
- [ ] Transparansi tidak menurunkan keterbacaan.
- [ ] Minimal satu scene dominan negative space.
