# Omniflow Design Style 9: Conversation Signal

**Cocok untuk:** Vecta AI, WhatsApp service, conversational sales, support automation.

## Creative Thesis

Percakapan adalah sinyal yang berkembang. Setiap pesan membawa intent, konteks, profil, dan tindakan berikutnya, bukan sekadar bubble balasan.

## Signature Device

**Signal Pulse:** waveform pesan berjalan antar-slide dan berubah menjadi knowledge match, profile completion, lead stage, next action, dan handoff.

## Palette

- Signal Black `#101416`
- Conversation Green `#38C878`
- Omniflow Blue `#2855FF`
- Intent Magenta `#E64BA8`
- Warm White `#F4F1E9`
- Waiting Gray `#667077`

## Typography

- Display: Syne
- Brand/body: Outfit
- Timestamp/metadata: IBM Plex Mono

## Material Grammar

- waveforms;
- transcript strips;
- typing cursor;
- context packets;
- intent labels;
- lead-stage pulse;
- handoff channel.

## Scene Types

1. Incoming Signal
2. Knowledge Match
3. Context Memory
4. Profile Capture
5. Intent Decode
6. Dual Output
7. Sales Handoff
8. Conversation Timeline

## Narrative Sequence

`MESSAGE → RETRIEVE → ANSWER → LEARN → CLASSIFY → ACT → HANDOFF → CONTINUE`

## Avoid

- Seluruh slide berupa chat bubble.
- Robot mascot.
- Klaim bebas halusinasi.
- Screenshot percakapan customer nyata.
- WhatsApp green menjadi background semua slide.

## QA

- [ ] Customer copy dan backend metadata dibedakan jelas.
- [ ] Signal Pulse hadir pada minimal enam slide.
- [ ] Handoff divisualkan sebagai tahap, bukan akhir otomatis.
- [ ] Teks chat cukup besar untuk feed.
- [ ] Tidak ada informasi pribadi nyata.
